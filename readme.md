    <p align="center">此项目是爬取阿里云漏洞平台构建漏洞预警，并将漏洞信息同步至飞书和邮箱</p>

- config.py 配置文件 可配置飞书API，邮箱等信息
- main.py 为启动文件


项目详情可参考：https://mp.weixin.qq.com/s/UXjZLHEiDwBHRQzAEkkxtg
### 运行方式

```bash
git clone https://github.com/oxff644/detector.git
cd detector/
bash run.sh
```
